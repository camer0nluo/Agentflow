# 全家第 2 階段需求—表單 UI 開發規範

## 表單大小

- A4 format: 1130 * 800 
- 長(1130px)須包含表單列印於A4上的範圍（不包含副本抄送元件、夾檔元件）
- 可視範圍: 0px ~ 副本元件位置
- Layout:

      ---------------- <=== 0
      |              |
      |              |
      |              |
      |   列印範圍   |
      |              |
      |              |
      |              |
      ---------------- <=== 1130
      |   夾檔元件   |
      ----------------
      |   副本抄送   |
      ---------------- <=== 可視範圍

*可使用工讀生表單開發Template*

## 部門章

- 電子簽章元件: 200px * 113px
